# Web eagxprague.org

## Lokální build

```
bundle # nainstaluje gemy
bundle exec jekyll serve
# otevři http://127.0.0.1:4000/
```

### Build JS

JS je v `js/main.js`.

`grunt uglify`

## Deploy

Je na to potřeba deploy klíč, který mají Michal a Dan.

```
./deploy.sh
```
